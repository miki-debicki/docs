# Getting Started with Managed Servers

Deploy custom MCP server code without managing infrastructure.

<Note>
**What you'll learn:**
- How to create a managed server
- Supported languages
</Note>

## Supported Languages

- TypeScript/JavaScript (Node.js)
- Python

## Creating a Managed Server

<Steps>
  <Step title="Navigate to Custom Servers">
    Go to **Servers** > **Custom Servers**.
  </Step>
  <Step title="Create Server">
    Click **New Custom Server** and select **Managed**.
  </Step>
  <Step title="Upload Code">
    Upload your MCP server code.
  </Step>
  <Step title="Deploy">
    Click **Deploy** and wait for the build to complete.
  </Step>
</Steps>
