## Types of Custom Servers

### Managed Servers

Upload your MCP server code and Metorial hosts it for you.

**Best for**: Custom business logic, internal APIs

### External Servers

Connect your self-hosted MCP server to Metorial.

**Best for**: On-premises requirements, existing infrastructure

### Forking

Clone and modify an existing integration.

**Best for**: Customizing existing integrations
